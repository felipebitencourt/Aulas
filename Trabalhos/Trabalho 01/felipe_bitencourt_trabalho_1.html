Perguntas sobre HTML

Felipe Bitencourt

1 - O que é HTML?

Uma linguagem de marcação para estruturar e definir o conteúdo de um site, premitindo a construção de websites e a inserção de novos conteúdos, como imagens, textos por meio dos hipertextos.

2 - O que é um elemento HTML? Dê exemplos.

Elemento é tudo que engloba o que está na tag; por exemplo a tag <img src="" alt=""> indica que será posto uma imagem no site, tendo como atributos: src="" e alt="" que indicam a origem da imagem e especifica um texto alternativo para a mesma, respectivamente. Tudo isso dentro da tag é chamado de elemento; sendo que o elemento pode conter textos, informações para o código, etc.

3 - Que ferramentas são necessárias para construir um site HTML?

Um editor de texto e navegador.

4 - Quais os elementos base necessários para a criação de uma página HTML?

Os elementos básicos são: <!DOCTYPE html> | <html> </html> | <head> </head> | <body> </body>.

5 - O elemento "br" necessita ser fechado?

Não.

6 - Para que servem os atributos? Dê um exemplo.

Os atributos impõe informações na tag para que a mesma se comporte da maneira indicada. Por exemplo, a tag <html lang="pt"> tem como atributo a parte lang="pt", que serve para indicar o idioma do website, neste caso, em português.
